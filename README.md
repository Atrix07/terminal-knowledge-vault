My Terminal Notes App

This is a simple program to save personal notes. You can add, view, search, edit, and delete notes. Notes are saved in a JSON file so your data stays even if you close the program.

(Run this in a terminal with Python 3, like PowerShell or the VS Code terminal.)

Things you can do:
Add new notes
View all notes with tags
Search notes by a word or tag
Edit notes if you want
Delete notes you dont need
Tags for notes 
export the notes

-How to run
Just follow the menu 1=view notes 2=Add note 3=Search note 4=Edit note 5=Delete note 6=Exit

-Sample notes.json

[
    {
        "id": 1,
        "title": "Phy assgn",
        "content": "quantum",
        "tags": [
            "phy",
            "assgn"
        ]
    },
    {
        "id": 2,
        "title": "ec project",
        "content": "12.12.25",
        "tags": [
            "ec"
        ]
    }
]


Each note has id,title,content and tags

*Can add tags to your notes to organize them and search easily. 

*There is an option to export all notes to a text file.
